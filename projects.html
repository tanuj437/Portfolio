Fitness Tracker

The Fitness Tracker System is a desktop application designed to enhance workout tracking by utilizing computer vision techniques. It accesses a webcam to analyze the user's exercise form, tracks metrics such as calories burned, exercise duration, and the number of sets and repetitions completed. The system provides real-time feedback to help users improve their posture and optimize their workouts.

Sentiment Analysis

The Sentiment Analysis Web Application is a tool that allows users to analyze the sentiment of textual data, such as product reviews or social media posts. It provides a simple interface where users can input text or upload CSV/Excel files for batch analysis. The application uses machine learning to classify the sentiment of the text into categories like positive, negative, or neutral, offering real-time feedback and allowing users to download the results. Built with Streamlit for the web interface and Scikit-learn for sentiment prediction, it efficiently processes and analyzes large volumes of text data.

QuizArt

QuizArt is an interactive quiz application that combines hand gesture recognition and creative drawing. Users can attempt quizzes by painting or drawing on a virtual screen using hand movements. The app tracks hand gestures to allow participants to select answers or draw their responses in an engaging, intuitive way. By leveraging computer vision techniques, the system detects hand motions to interact with the quiz interface, making the quiz-taking experience both fun and interactive. This project integrates gesture recognition, real-time feedback, and creative participation, offering a unique blend of education and entertainment.

HealthOptimized

HealthOptimized is a predictive healthcare application that helps users assess their risk of various diseases, such as diabetes, heart disease, and others, based on personal health data. Users fill out a form with relevant health information, including lifestyle factors, medical history, and symptoms. The app uses machine learning algorithms to analyze this data and predict the likelihood of developing certain conditions. With a user-friendly interface, HealthOptimized provides instant, personalized health insights, helping users take proactive steps toward maintaining their health and preventing potential risks.
